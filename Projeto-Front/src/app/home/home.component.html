Como fazer parte do Clippy?

Primeiro passo: Cadastro

Primeiro você inicia se cadastrando no Clippy clicando aqui. Caso já cadastrado efetue seu login.

Segundo passo: Verificação

Para manter o objetivo do Clippy, todos os novos conteúdos publicados são verificados, para confirmarmos se a nova postagem aborda a temática sócio ambiental, mantendo assim o Clippy uma rede fluida de informação.

Terceiro passo: Postagem

Agora com o seu post verificado ficará postado no seu feed para que você e os demais usuários possam visualizar.
